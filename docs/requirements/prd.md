# 产品需求文档

## 1. 项目背景

AI 编程工具正在快速发展，但大部分用户仍停留在"对话式提问"的使用方式。本书旨在系统化地介绍如何通过 OpenCode 生态构建高效的 AI 编程工程流水线。

## 2. 项目目标

- 提供完整的 Harness Engineering 知识体系
- 包含可操作的最佳实践和可复用的配置
- 保持与 OpenCode 生态的同步更新
- 开源社区协作模式

## 3. 功能需求

### 3.1 书籍内容

- 7 章主体内容，20+ 篇文章
- 每章包含概念讲解、实战示例、最佳实践

### 3.2 配套资源

- 可运行的 OpenCode 配置示例
- 示例 Skill 代码
- GitHub Actions 部署流水线

### 3.3 技术指标

- 支持 GitHub Pages 访问
- 支持本地 docsify 预览
- 响应式设计，支持移动端阅读
