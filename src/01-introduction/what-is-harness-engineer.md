## 从 Copilot 到 Harness Engineer

AI 编程工具的发展经历了三个阶段：

1. **代码补全时代** — GitHub Copilot、TabNine
2. **对话编程时代** — Cursor、Claude Code
3. **工程流水线时代** — OpenCode、Agent 编排

Harness Engineer 就是第三时代的核心角色。

## 核心能力

- **环境搭建** — 配置和优化 AI 编程环境
- **工作流设计** — 设计高效的 Agent 协作流程
- **Skill 开发** — 编写可复用的能力模块
- **持续改进** — 基于实际反馈不断优化流水线

## 与其他角色的区别

| 角色 | 关注点 | AI 使用方式 |
|------|--------|------------|
| 普通开发者 | 写代码 | 对话式提问 |
| AI Engineer | 训练/调优模型 | 模型开发 |
| **Harness Engineer** | **工程流水线** | **Agent 编排** |
