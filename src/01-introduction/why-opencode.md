## 与其他工具的对比

| 特性 | OpenCode | Cursor | Claude Code |
|------|----------|--------|-------------|
| 开源 | ✓ | ✗ | ✗ |
| 多 Provider | ✓ | 有限 | ✗ |
| Plugin 系统 | ✓ | ✗ | ✗ |
| Skill 系统 | ✓ | ✗ | ✗ |
| 本地模型 | ✓ | ✗ | ✗ |

## 核心优势

1. **完全开源** — 可审计、可定制、可贡献
2. **Provider 无关** — 不绑定任何模型服务商
3. **Plugin 生态** — 通过 oh-my-openagent 等插件无限扩展
4. **Skill 系统** — 将最佳实践封装为可复用的"思想武器"
